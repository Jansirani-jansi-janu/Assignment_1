Local Food Wastage Management System

This project aims to reduce food waste and support underprivileged communities by creating a digital platform that connects food providers (such as restaurants, grocery stores, and individuals) with food receivers (such as NGOs or needy individuals).
The system was developed using Python and Streamlit for the frontend, and SQL for the backend database. It includes features like:

User registration and login for providers and receivers

Posting and claiming of leftover or surplus food

Admin panel to monitor and manage data

15+ SQL analytical queries to generate insights, such as:

Most donated food types

Top providers and receivers

Unclaimed food statistics

The interface is user-friendly and allows real-time tracking of donations, helping reduce waste and make efficient use of excess food.
Frontend: Streamlit

Backend: Python

Database: SQL

Platform: Google Colab
